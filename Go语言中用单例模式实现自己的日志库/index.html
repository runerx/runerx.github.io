Go语言中用单例模式实现自己的日志库

发表于 2022-12-06 分类于 Code 阅读次数：

Go语言自带的日志库不能调整日志输出级别，其他日志库需要通过实例的地址传递实现配置的一致性。对于一个日志组件来说，我们最好全局维护同一个实例。这样刚好满足单例模式。单例模式是常用的模式之一，在它的核心结构中只包含一个被称为单例的特殊类，能够保证系统运行中一个类只创建一个实例。我们就用单例模式来封装一个日志库。
