2. linux内核源码目录介绍

下载完后我们可以进入目录

arch目录

arch目录是CPU体系相关的代码，有arm架构、x86架构等等。

每个架构下面还有内核相关代码

Documentation

内核里的技术文档，关于内核参数、配置、特性等都有

firmware

固件相关代码

init

init是内核启动相关的代码

main.c里面有一个start_kernel函数，这是开始执行C代码的位置，在此之前都是汇编代码。

block

block是块设备相关代码

drivers

drivers代码量很大，外设特别多，所以需要更多的驱动支持。

ipc

ipc是进程通讯相关代码

security

安全相关代码

net

协议栈相关代码

sound

声音相关代码

fs

文件系统相关代码

kernel

kernel是内核核心相关代码，进程管理、进程调度等内核核心代码

include

头文件代码

mm

内存管理相关代码

*表示把驱动编到内核里面

配置好后选择Save,再选择Exit退出，
